categorization of javascript operators based on their functionality

In JavaScript, operators can be categorized based on their functionality. Here are the main categories: 1:- Arithmetic Operators: Perform mathematical operations on numeric values. Examples: + (addition), - (subtraction), * (multiplication), / (division), % (modulus), ++ (increment), -- (decrement).
example: let a = 5; let b = 2; console.log(a + b); // Addition: 7 console.log(a - b); // Subtraction: 3 console.log(a * b); // Multiplication: 10 console.log(a / b); // Division: 2.5 console.log(a % b); // Modulus: 1 console.log(++a); // Increment: 6 console.log(--b); // Decrement: 1 2:- Assignment Operators: Assign values to variables. Examples: = (assignment), +=, -=, *=, /= (compound assignment).
example: et x = 10; x += 5; // Same as x = x + 5 console.log(x); x -= 3; // Same as x = x - 3 console.log(x); x *= 2; // Same as x = x * 2 console.log(x); x /= 4; // Same as x = x / 4 console.log(x); 3:- Comparison Operators: Compare two values and return a Boolean result. Examples: == (equality), === (strict equality), != or !== (inequality), > (greater than), < (less than), >= (greater than or equal to), <= (less than or equal to).
exapmle: let p = 5; let q = "5"; console.log(p == q); console.log(p === q); console.log(p != q); console.log(p !== q); console.log(p > 3); console.log(p < 3); console.log(p >= 5); console.log(p <= 4); 4:- Logical Operators: Perform logical operations on Boolean values. Examples: && (logical AND), || (logical OR), ! (logical NOT).
example: let isTrue = true; let isFalse = false; console.log(isTrue && isFalse); console.log(isTrue || isFalse); console.log(!isTrue); 5:- Bitwise Operators: Perform bitwise operations on integer values. Examples: & (bitwise AND), | (bitwise OR), ^ (bitwise XOR), ~ (bitwise NOT), << (left shift), >> (right shift), >>> (unsigned right shift).


example: let m = 5; let n = 3; console.log(m & n); console.log(m | n); console.log(m ^ n); console.log(~m); console.log(m << 1); console.log(m >> 1);
